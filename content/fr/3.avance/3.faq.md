## Questions fréquentes

**Mes commissions s'affichent en négatif, est-ce normal ?**

Non, les commissions doivent être des valeurs positives. Utilisez le bouton **"Fix"** dans la console LogView pour corriger les valeurs existantes, ou réimportez vos trades avec la dernière version de l'application.

---

**Puis-je importer des trades d'options ?**

Oui, via l'import CSV personnalisé. Utilisez le symbole complet de l'option dans la colonne `symbol`.

---

**Comment supprimer un trade ?**

Ouvrez le trade via le [calendrier](/fr/guide/calendrier), puis cliquez sur **"Supprimer"** et confirmez.

---

**Puis-je utiliser TradeJourney hors ligne ?**

Oui, TradeJourney fonctionne entièrement en local. Aucune connexion Internet n'est requise après l'installation.

---

**Les données sont-elles sécurisées ?**

Oui, toutes vos données restent sur votre machine dans des fichiers SQLite locaux. Rien n'est envoyé vers des serveurs externes.

---

**Puis-je avoir plusieurs bases de données ?**

Oui, vous pouvez créer autant de bases que nécessaire — utile pour séparer différentes périodes, stratégies ou comptes.

---

**Comment accéder à TradeJourney depuis mon téléphone ?**

Assurez-vous que votre téléphone est sur le même réseau Wi-Fi, puis accédez à `http://[IP-de-votre-machine]:3000`. Trouvez votre IP avec `ip addr` ou `ifconfig`.

---

**Comment faire une sauvegarde de mes données ?**

Copiez simplement le dossier `data/` :

```bash [Terminal]
cp -r data/ backup/data-$(date +%Y%m%d)/
```

Consultez la page [Filtres et export](/fr/guide/filtres-export) pour plus d'options.
