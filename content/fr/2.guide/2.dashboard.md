Le dashboard centralise toutes vos métriques de performance et graphiques.

## Métriques principales

### PnL Cumulé

Évolution de votre capital dans le temps. Si un capital de départ est configuré, le graphique part de ce montant.

### Win Rate

Pourcentage de trades gagnants.

```text
Win Rate = (Trades gagnants / Total trades) × 100
```

### Profit Factor

Ratio entre les gains totaux et les pertes totales.

```text
Profit Factor = Total gains / Total pertes
```

| Valeur | Interprétation     |
| ------ | ------------------ |
| < 1    | Stratégie perdante |
| 1 – 2  | Acceptable         |
| > 2    | Excellent          |

### Sharpe Ratio

Performance ajustée au risque. Mesure la régularité des gains.

| Valeur | Interprétation |
| ------ | -------------- |
| > 1    | Bon            |
| > 2    | Excellent      |

### Recovery Factor

Capacité à récupérer après une perte.

```text
Recovery Factor = Profit net / Drawdown maximum
```

### Expectancy (APPT)

Gain moyen attendu par trade.

```text
APPT = (Win Rate × Gain moyen) - (Loss Rate × Perte moyenne)
```

## Graphiques disponibles

- **PnL Cumulé** : Évolution du capital dans le temps
- **PnL par jour** : Performance quotidienne (barres vertes/rouges)
- **Win Rate** : Évolution du taux de réussite
- **APPT** : Expectancy dans le temps

## Filtres du dashboard

Affinez vos analyses avec les filtres disponibles :

- **Période** : Aujourd'hui, Cette semaine, Ce mois, Personnalisé
- **Comptes** : Un ou plusieurs comptes de trading
- **Agrégation** : Par jour, semaine ou mois

## Comparer plusieurs comptes

Vous pouvez voir les performances :

- **Consolidées** : Tous les comptes agrégés
- **Par compte** : Sélectionnez un compte spécifique dans le filtre
- **En comparaison** : Sélectionnez plusieurs comptes pour les comparer
