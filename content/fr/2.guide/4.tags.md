Les tags permettent d'organiser et d'analyser vos trades par catégorie.

## Créer des groupes de tags

1. Allez dans **Paramètres** → **Tags**
2. Créez des groupes de tags
3. Ajoutez des tags dans chaque groupe

**Exemples de groupes :**

::card-group
  :::card
  ---
  icon: i-lucide-brain
  title: Stratégie
  ---
  Breakout, Pullback, Reversal
  :::

  :::card
  ---
  icon: i-lucide-crosshair
  title: Setup
  ---
  Morning Range, Afternoon Trend, News
  :::

  :::card
  ---
  icon: i-lucide-globe
  title: Marché
  ---
  Forex, Indices, Commodities
  :::

  :::card
  ---
  icon: i-lucide-star
  title: Qualité
  ---
  A+, A, B, C
  :::
::

## Assigner des tags à un trade

1. Ouvrez le trade depuis le [calendrier](/fr/guide/calendrier)
2. Sélectionnez les tags appropriés dans les groupes
3. Cliquez sur **Sauvegarder**

## Analyser par tags

Le dashboard vous permet de :

- **Filtrer** les métriques par tag ou groupe de tags
- **Comparer** les performances entre différents tags
- **Identifier** vos setups les plus rentables
- **Analyser** le win rate par stratégie

## Filtrer par tags

Dans le dashboard ou la liste des trades, utilisez le filtre **Tags** pour sélectionner un ou plusieurs tags. Les métriques se recalculent automatiquement sur la sélection filtrée.
