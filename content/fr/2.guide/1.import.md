TradeJourney supporte plusieurs formats d'import pour s'intégrer avec votre plateforme de trading.

## Import depuis MT5 (MetaTrader 5)

::steps
### Exporter depuis MT5

Dans MT5, allez dans **Historique des comptes** → clic droit → **Rapport** → **Enregistrer en HTML**.

### Importer dans TradeJourney

1. Allez dans **Import** → **MT5**
2. Sélectionnez le fichier HTML exporté
3. Configurez le fuseau horaire
4. Cliquez sur **Importer**
::

**Format attendu :**

Le fichier HTML doit contenir les colonnes : `Time`, `Position`, `Symbol`, `Type`, `Volume`, `Price`, `S/L`, `T/P`, `Commission`, `Swap`, `Profit`.

## Import depuis NinjaTrader

::steps
### Exporter depuis NinjaTrader

Exportez vos trades depuis l'interface NinjaTrader.

### Importer dans TradeJourney

1. Allez dans **Import** → **NinjaTrader**
2. Sélectionnez le fichier exporté
3. Vérifiez la correspondance des colonnes
4. Cliquez sur **Importer**
::

## Import CSV personnalisé

Pour les autres plateformes, utilisez l'import CSV générique.

### Format du fichier CSV

Préparez un fichier avec les colonnes suivantes :

| Colonne      | Description          | Format          |
| ------------ | -------------------- | --------------- |
| `symbol`     | Symbole tradé        | Ex: `EURUSD`    |
| `openDate`   | Date d'ouverture     | ISO 8601        |
| `closeDate`  | Date de fermeture    | ISO 8601        |
| `type`       | Direction            | `buy` ou `sell` |
| `lot`        | Taille de position   | Nombre décimal  |
| `openPrice`  | Prix d'entrée        | Nombre décimal  |
| `closePrice` | Prix de sortie       | Nombre décimal  |
| `profit`     | Profit brut          | Nombre décimal  |
| `commission` | Commission (positif) | Nombre décimal  |
| `swap`       | Frais de swap        | Nombre décimal  |

**Exemple :**

```csv [trades.csv]
symbol,openDate,closeDate,type,lot,openPrice,closePrice,profit,commission,swap
EURUSD,2024-01-15T09:30:00Z,2024-01-15T14:20:00Z,buy,0.1,1.0850,1.0870,20.00,2.50,0
GBPUSD,2024-01-16T10:00:00Z,2024-01-16T15:30:00Z,sell,0.2,1.2650,1.2630,40.00,5.00,0
```

### Procédure d'import

1. Allez dans **Import** → **CSV**
2. Sélectionnez votre fichier
3. Mappez les colonnes si nécessaire
4. Cliquez sur **Importer**

## Gestion des fuseaux horaires

TradeJourney gère les fuseaux horaires de trois façons :

- **Mode UTC** : Les dates sont converties en UTC
- **Mode Local** : Les dates sont conservées en heure locale
- **Par compte** : Un fuseau horaire spécifique peut être configuré par compte dans **Paramètres** → **Comptes**

## Problèmes courants

::callout{color="orange" icon="i-lucide-triangle-alert"}
**Commissions négatives** : Les commissions doivent être des valeurs positives. Si elles apparaissent en négatif, utilisez le bouton **"Fix"** dans la console LogView, ou réimportez avec la dernière version.
::

- **Import MT5 échoue** : Vérifiez que le fichier est bien en format HTML et contient les colonnes attendues. Essayez un fuseau horaire différent.
- **Import CSV échoue** : Vérifiez l'encodage du fichier (UTF-8 recommandé) et que les dates sont au format ISO 8601.
