## Première connexion

Après avoir démarré l'application, rendez-vous sur **<http://localhost:3000>**.

::steps
### Créer votre compte

Cliquez sur **"S'inscrire"**, remplissez vos informations et validez.

### Créer une base de données

TradeJourney utilise SQLite pour stocker vos données localement. Chaque base de données est indépendante.

1. Cliquez sur **"Nouvelle base de données"**
2. Donnez un nom (ex: `Trading 2024`)
3. La base est créée automatiquement

  :::callout{icon="i-lucide-lightbulb"}
  Vous pouvez créer plusieurs bases pour séparer différentes périodes ou stratégies.
  :::

### Créer un compte de trading

1. Allez dans **Paramètres** → **Comptes**
2. Cliquez sur **Nouveau compte**
3. Remplissez les champs :

- **Nom** : Identifiant unique (ex: `MT5-123456`)
- **Nom affiché** : Nom convivial (ex: `Compte Principal`)
- **Capital de départ** : Montant initial (optionnel)

### Importer vos premiers trades

Rendez-vous dans la section [Import de trades](/fr/guide/import) pour importer depuis MT5, NinjaTrader ou CSV.
::

## Capital de départ

Le capital de départ permet d'afficher votre PnL cumulé depuis votre capital initial :

| Configuration               | Résultat sur le graphique                         |
| --------------------------- | ------------------------------------------------- |
| Sans capital                | Le graphique part de 0                            |
| Avec capital (ex: 10 000 €) | Le graphique part de 10 000 € et affiche le total |

**Exemple :** Capital de départ 10 000 € + PnL cumulé +2 500 € = graphique affiche 12 500 €
