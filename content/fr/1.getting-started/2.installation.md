## Prérequis

Avant de commencer, assurez-vous d'avoir :

- **Docker** et **Docker Compose** installés (recommandé)
- OU **Node.js 18+** pour une installation manuelle
- Au moins **500 MB** d'espace disque
- Un navigateur moderne (Chrome, Firefox, Safari, Edge)

## Installation avec Docker (Recommandé)

::steps
### Cloner le repository

```bash [Terminal]
git clone https://github.com/yourusername/tradeJourney.git
cd tradeJourney
```

### Configurer l'environnement

```bash [Terminal]
cp .env.example .env
```

Configuration minimale dans `.env` :

```bash [.env]
PORT=3000
DEBUG_MODE=false
MAX_FILE_SIZE=10485760
```

### Démarrer l'application

```bash [Terminal]
# Mode développement
docker-compose -f docker-compose.dev.yml up -d

# OU Mode production
docker-compose up -d
```

### Accéder à l'application

Ouvrez votre navigateur sur **http://localhost:3000** puis créez votre compte et votre première base de données.
::

## Installation manuelle (sans Docker)

::steps
### Vérifier les prérequis

```bash [Terminal]
node --version  # >= 18
npm --version
```

### Cloner et installer

```bash [Terminal]
git clone https://github.com/yourusername/tradeJourney.git
cd tradeJourney
npm install
```

### Configurer l'environnement

```bash [Terminal]
cp .env.example .env
```

### Initialiser les données

```bash [Terminal]
mkdir -p data
```

### Démarrer l'application

```bash [Terminal]
# Mode développement
npm run dev

# OU Mode production
npm run build && npm run start
```
::
