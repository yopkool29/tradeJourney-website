Bienvenue sur **TradeJourney**, un journal de trading local et open-source qui vous permet d'analyser vos performances sans envoyer vos données vers un serveur externe.

## Qu'est-ce que TradeJourney ?

TradeJourney est une application web auto-hébergée conçue pour les traders qui souhaitent :

- **Importer** leurs trades depuis leur plateforme de trading (MT5, NinjaTrader, CSV)
- **Analyser** leurs performances avec des métriques professionnelles
- **Organiser** leurs trades avec un système de tags et de comptes
- **Visualiser** leur progression via un calendrier et des graphiques
- **Garder le contrôle** de leurs données — tout reste en local sur leur machine

## Fonctionnalités principales

- **Import multi-sources** : MetaTrader 5 (HTML), NinjaTrader, CSV personnalisé
- **Dashboard complet** : PnL cumulé, Win Rate, Profit Factor, Sharpe Ratio, Recovery Factor, Expectancy (APPT)
- **Calendrier de trading** : Vue jour par jour avec intensité proportionnelle au PnL
- **Système de tags** : Organisez par stratégie, setup, marché ou qualité
- **Multi-comptes** : Gérez plusieurs comptes de trading et comparez leurs performances
- **Multi-bases de données** : Séparez vos périodes ou stratégies dans des bases SQLite distinctes
- **Screenshots** : Attachez des captures d'écran à vos trades pour les revoir
- **Export** : Exportez vos données en CSV ou JSON
- **100% local** : Aucune donnée envoyée, aucun cloud, aucun abonnement

## Technologies utilisées

TradeJourney est construit avec :

- **Nuxt 3** pour le frontend
- **SQLite** pour le stockage local des données
- **Docker** pour un déploiement simplifié (optionnel)
